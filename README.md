## Usage

```Bash
usage: main.py [-h] [-w] [-i I] [-f F] [-t T]

Hide message in png file .

optional arguments:
  -h, --help  show this help message and exit
  -w          write a message in picture
  -i I        input file
  -f F        file with text
  -t T        msg to write
  ```
